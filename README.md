# Nodejs-project
 # Flight Management System ✈️

A complete web-based Flight Management System built with HTML, CSS, and Node.js.

## ✨ Features

- **Add Flights** - Add new flights with validation
- **View Flights** - See all flights in a table
- **Update Status** - Mark flights as departed
- **Sample Data** - Comes with example flights
- **Responsive** - Works on mobile and desktop

## 📋 Requirements

- Node.js (v14 or higher)
- npm (comes with Node.js)
